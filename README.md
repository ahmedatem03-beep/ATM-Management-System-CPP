# 🏧 ATM Management System (C++)

A console-based **ATM Management System** developed using **C++** that simulates the basic functionalities of a real ATM machine.

The project allows users to securely log in, perform banking transactions, and manage their accounts through an interactive console interface.

---

## 📌 Features

- 🔐 User Login Authentication
- 💰 Check Account Balance
- 💵 Deposit Money
- 💸 Withdraw Money
- 🔄 Quick Withdraw
- 📊 View Account Information
- 💾 File Handling for Data Persistence
- 🚪 Logout and Exit

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Visual Studio
- Windows Console
- Standard Template Library (STL)
